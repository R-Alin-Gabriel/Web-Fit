# Fitness Website

#### I have built a one-page responsive Fitness website using: 
* HTML
* CSS
* JavaScript 
#### Compatible with all mobile devices 

### Features 
* One Page Layout
* Custom Icons
* Drop Down Hamburger Menu
* On-hover Effect
* Responsive Layout
* Social Media Icons
* Email Subscription
* Rich Footer Available

### Technologies

#### Project is created with:
* Bootstrap@5.0.2
* Fontawesome v5.10.0
* Pictures from https://www.pexels.com/

### Files Included

* index.html
* styles.css
* app.js
* img folder
